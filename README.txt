ECO TONEL - painel por áreas

Arquivos principais:
- index.html -> login
- gestao.html -> gestão geral
- operacao.html -> operação e logística
- administrativo.html -> administrativo e regularização
- financeiro.html -> financeiro e precificação
- comercial.html -> comercial e atendimento

Como testar:
1. Extraia a pasta.
2. Abra o arquivo index.html no navegador.
3. Use um dos acessos abaixo:
   - gestao / 123456
   - operacao / 123456
   - administrativo / 123456
   - financeiro / 123456
   - comercial / 123456

Observação:
- Esta versão funciona localmente com armazenamento no navegador (localStorage).
- Para restaurar os dados padrão, use o botão "Restaurar dados demo" na área logada.
- Os arquivos de identidade visual nova já estão aplicados em assets/img.
